Проект написан на **Turbo Pascal 7.0**.

Исходный код сохранён в файле `.pas`.

## Используемая среда
- Turbo Pascal 7.0
- DOSBox

## Как запустить / скомпилировать
1. Запустить DOSBox
2. Смонтировать папку с проектом
3. Открыть `TP.EXE`
4. Открыть файл `.pas`
5. Нажать `Alt + F9` для компиляции
